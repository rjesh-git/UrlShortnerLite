### Summary ###
Generates short urls for the document library app contents. This add-in works in: SharePoint 2013 or later.

### Applies to ###
-  Office 365 Multi Tenant (MT)
-  Office 365 Dedicated (D)
-  SharePoint 2013 on-premises
